Introduction

Javascript (à ne pas confondre avec java) est un langage de programmation dynamique et non typée.(c'est à dire qu'une variable peut contenir n'importe quel type de valeur)Le JS est utilisé le plus souvent pour le coté client mais peut être utilisé aussi coté serveur (nodeJS).
Javascript est communément appelé JS, elle est sensible à la casse, cela signifie que le langage considére les majuscules différents des miniscules.Les mots clés en JS sont tous en miniscules.
Javascript a été crée en 1995 par BrendanEich et intégré au navigateur Netscape navigator.
Les fichiers JS seuls ne peuvent pas être exécutés par le navigateur, il faut donc les intégrer à une page HTML.

2. Premier pas

2.1 Javascript dans le fichier HTML

Dans un fichier .html, les codes JS peuvent être écrits à l'intérieur des balises script qu'on place généralement dans le head accompagné de l'attribut defer ou tout en bas de la balise body.
(voir Javascript.html)

2.2 Javascript dans le fichier externe

Les codes JS sont écrits dans un fichier externe généralement nommé script.js. Aucune balise ne sera utilisé dans se fichier., il ne contiendra que du code JS.
Par contre il sera nécessaire d'importer se fichier dans notre document HTML pour pouvoir l'éxecuter.
(voir Javascript.html)

2.3 Les commentaire en Javascript

Les commentaires sont des lignes de textes placées dans le code et servant à documenter ce dernier, c'est à dire à expliquer se que fait tel ou tel bout de script et éventuellemnt comment la manipuler.
Un commentaire est donc une indication qui ne sera pas lu par le navigateur, donc invisible pour les utilisateurs. Pour écrire un commentaire monoligne en Js, on utilise le double slash (//).
Pour les commentaires multilignes, on utilise slash et étoile pour entourer le texte du commentaire. *commentaires*/

3.Les variables

Définition

Comme vu précédement en algorithmie, une variable est un conteneur servant à stocker des informations de maniére temporaire. Le propre d'une variable comme sont nom l'indique est donc de varier, c'est à dire de pouvoir contenir plusieurs valeurs au fil du temps.
C'est bien cet aspect qui les rendent utiles cependant il faut bien différencié la variable q'on appel par son nom de la valeur qu'elle contienne. Il n'y a pas d'égalité entre la variable et sa valeur meme si la synthaxe porte à confusion.

3.2 Déclaration

Pour déclarer une varaible en JS, on utilise les mots clé let ou var suivi du nom de la variable.
Le nom d'une variable doit obligatoirement commencer par une lettre ou un underscore(_) et ne doit jamais commencer par chiffre.
Le nom d'une variable ne doit pas contenir d'espace mais peut contenir des lettres, des chiffres, des tirets mais pas de caractéres spéciaux.
En JS, il est recommandé d'utiliser le camelCase pour nommer les variables.

    exemple:
  • let puitDufou= "parc d'attraction";

3.3 Les types de variables

En JS, on distingue 3 types de variables.
Numbers: ce sont des variables de types numériques.
String: ce sont des variables de types chaines de caractéres.
Boolean: ce sont des variables de types True ou False.

3.4 Les types de valeurs

En JS, contrairement à la plus part des langages de programmation, nous n'avons pas besoin de préciser le type de variable à la déclaration. Le JS étant un langage non typé , c'est la valeur qui permet de déterminer le type de la variable.

    Ci-dessous les types de valeurs:
  • String
  • Number
  • Boolean
  • Null
  • Object
  • Undefined

NB : On peut utiliser la fonction prédéfini typeof pour déterminer le type de valeur ou de variable.
voir javascript.html

C'est comme la condition alternative si, sinon.

7. Les chaines de caractéres

Définition

JS propose plusieurs méthodes pour les chaines de caractéres y compris des méthodes de style.

    exemple:
    document.write(chaine.toLowerCase(), " (toLowerCase)", )//min;

9. Les boucles

9.1 La boucle for

L'instruction for permet de créer une boucle avec 3 éléments séparé par des ';' et entouré par des parenthéses suivi d'une paire d'accolades contenant l'ensemble des instructions à éxecuter.
Le 1er élément est la condition de départ; le 2nd, la condition d'arrêt et le dernier est le pas.
Voir javascript.html

9.1.1 Les break et continue

Les instructions break et continue sont trés important pour manipuler les boucles.
Le break permet d'arrêter la boucle à l'itération ciblée.
Le continue permet de sauter l'itération ciblée et de poursuivre la boucle par la suite.
Voir javascript.html

9.2 La boucle for in

L'instruction for in permet d'itérer les indices assocciées aux valeurs d'un tableau.

9.3 La boucle for of

L'instruction for of permet d'itérer les valeurs d'un tableau.

9.4 La boucle for each

L'instruction des forEach permet d'itérer en même temps les indices et les valeurs d'un tableau.

9.5 La boucle while

L'instruction while permet de créer une boucle, elle teste d'abofrd une condition et éxecute les instructions tant que la condition est vérifiée.
Voir javascript.html

9.6 La boucle do while

L'instruction do while permet de créer une boucle qui exécute tant que la condition est vérifiée. Elle éxecute d'abord les instruction ensuite, teste la condition. Se qui veut dire qu'avec le do while, les instructions s'exécutent au moins une fois.
Voir javascript.html

10. Les tableaux

10.1 Les tableaux à une dimension

Un tableau est un conteneur qui peut contenir plusieurs valeurs de type différent indépendantes ou liées entre elle.
Les différentes valeurs d'un tableau sont stockées dans des cases qu'on identifie avec des incices qui vont de 0 au nombre d'élément du tableau. Un tel tableau est appelé tableau indéxé.
Dans la programmation orineté objet, un tableau hérite de la class array donc un tableau est un objet array.
Voir javascript.html

10.2 Les méthodes array

    exemple:
  • tab1.pop();//supprime le dernier élément du tableau
  • tab1.shift();//supprime le 1er élément
  • tab1.push("Camil");//ajoute 1 élément à la fin du tab
  • tab1.unshift("Janat");//ajoute 1 élément au début du tab
  • tab1.splice(2,0,"Boris","Jacques","Marie","Paulette","Jade");//ajoute des éléments à partir d'un indice ciblé du tab
  • tab1.splice(2,2);//supprime 2 éléments à partir d'un indice ciblé du tab

10.3 Les tables multidimensionnels

Comme aborder en algo, un tableau multidimensionnel n'est rien d'autre qu'un tableau de tableau.
Les mêmes principes sont applicable aussi bien au tableau simple qu'aux tableaux multidimensionnels. Cependant on aura besoin d'autant de boucle que de tableau pour parcourir un tableau multidimensionnel.
Voir javascript.html

11. Les fonctions

Définition

Une fonctionest un groupe de ligne de code de programmation, écrite par un concepteur et destinées à exécuter une tâche bien spécifique.
On appelle fonction un sous-programme qui permet d'éffectuer un ensemble d'instruction par simple appel de la fonction dans le corps du programme principal.
Une fonction est un groupe de ligne de code de programmation destiné à éxecuter une tâche bien spécifique et que l'on pourra utilisier à plusieurs reprises. En JS, il existe 2 types de fonctions: Les fonctions prédéfinies et les fonctions déclarées.

11.2 Les fonctions prédéfinies

En JS, il existe une certains nombres de fonctions prédéfinies, elles sont souvent associées à un objet c'est pour ce là qu'on les appelle méthodes.
Les fonctions alert(),prompt(),push(),shift()... sont quant à elles des méthodes de l'objet Array.

11.3 Les fonctions déclarées

Elles sont les fonctions mises en place par le programmeur selon les besoins de l'application our effectuer une tâche bien précise.

    Synthaxe:
  • function nom_de_la_fonction(arguments){---instruction---}
    Fonction fléchées synthaxe:
  • (arguments) => {---instruction---}

Appel d'une fonction:

Pour appeler la fonction, il faut bien sûr qu'elle ait été définie auparavant. Par conséquent, il est conseillé de commencer la définition de la fonction et les appels dans le corps de la page. L'appel d'une fonction se fait par son identifiant suivi impérativement de parenthèses, lesquelles peuvent contenir les arguments à faire passer.
Peyt avoir lieu à n'importe quel endroit de la page :
-dans d'autres fonction,
-dans le corps de la page.

12. Les objets

JS est conçu autour d'un paradigme simple basé sur les objets. Un objet est un ensemble de propriétés et de méthodes. Une propriété est une association entre un nom(auusi appelé clé) et une valeur. La valeur d'une propriété peut être appelé "méthode". En plus des objets natifs fournis par l'environnement, il est possible de construire ses propres objets.
JS n'utilisait pas d'instruction de class comme d'autre langages de programmation orientée Objet comme java,c++,...Il utilisait les fonctions comme instruction de class.
Mais depuis ES6(ECMAScript)le standard qui définit les normes de javascripst publié en 2015, l'instruction class est présente et permet de construire des objets.
_A l'intérieur de la fonction ou de la classs, on utilise: this.propriété.
_A l'extérieur de la fonction ou de la class, on utilise: instance.propriété.
